The methods of the IniFile class are listed below. For a complete list of IniFile class members, see the IniFile Members topic.

Public Instance Methods

AddKey Adds a key/value to a given section. If the section does not exist, it is created.
AddSection Adds a section to the IniFile. If the section already exists, then no action is taken.
ChangeValue Changes the value of a given key in a given section. Both the key and the section must exist.
CommentKey Comments a given key in a given section. Both the key and the section must exist.
CommentSection Comments a given section, including all of the keys contained in the section.
DeleteKey Deletes a given key in a given section. Both the key and the section must exist.
DeleteSection Deletes a given section. The section must exist. All the keys in the section will also be deleted.
