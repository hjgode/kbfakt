IniFile.AddKey Method 

Adds a key/value to a given section. If the section does not exist, it is created.

Parameters

KeyName
The name of the key to add. If the key alreadys exists, then no action is taken.
KeyValue
The value to assign to the new key.
SectionName
The section to add the new key to. If it does not exist, it is created.
IsCommented
Optional, defaults to false. Will create the key in commented state.
InsertBefore
Optional. Will insert the new key prior to the specified key.

Return Value

Remarks

If the section does not exist, it will be created. If the 'IsCommented' option is true, then the newly created section will also be commented. If the 'InsertBefore' option is used, the specified key does not exist, then the new key is simply added to the section. If the section the key is being added to is commented, then the key will be commented as well.
